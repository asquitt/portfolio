# Demario Portfolio — Apple-Style One Page

Ready-to-run Vite + React + Tailwind + Framer Motion portfolio.

## Quick Start

```bash
npm install
npm run dev
```

Then open the printed localhost URL.

## Deploy
- **CodeSandbox**: File → Import Project → Upload this folder/zip.
- **Vercel**: vercel deploy (or import the GitHub repo).

## Customize
- Edit text in `src/App.jsx` (sections, experience, projects).
- Replace `resume.pdf` with your real resume.
- Tweak styles in `tailwind.config.js` and `src/index.css`.
